# Telegram Bot для чек-листов смены

Бот помогает сотрудникам кофейни проходить чек-листы открытия и закрытия смены.

## Установка и запуск

1. Клонируйте репозиторий
2. Создайте виртуальное окружение и активируйте его
3. Установите зависимости:
 pip install -r requirements.txt
4. Добавьте токен бота в файл `.env`  ( пример см. `.env.example`)
5. Запустите бота:

## Команды

- `/start` — начать работу с ботом
- `/cancel` — отменить текущий диалог
